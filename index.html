    Predictive Share Pricing

    This project, StockVision, is designed to forecast stock prices by implementing multiple machine learning regression models. Using scikit-learn, it builds predictive models from historical stock data, with each model capturing distinct patterns to offer a robust forecast across various stocks. The workflow begins with data preprocessing, where historical price data undergoes feature engineering to include temporal information (year, month, day, day of the week) and lag-based features, capturing recent closing prices as predictive inputs. These enhanced features provide the models with both temporal and sequential price information.
    Five machine learning models: Linear Regression, Ridge Regression, Lasso Regression, Random Forest Regressor, and Gradient Boosting Regressor, are trained and evaluated using Mean Squared Error (MSE) and RÂ² metrics. Each model's performance is assessed on a test set to understand its predictive accuracy and generalization ability. The Random Forest and Gradient Boosting Regressors, with their capacity for handling non-linearity, often show superior performance in capturing price trends and volatility. For forecasting, the project iteratively uses the best-performing model, updating lag features based on the latest forecasted price to generate sequential future predictions. This iterative approach captures temporal dependencies in stock movements and offers a dynamic prediction that adapts as new data points emerge.
    As of now, this project is in its early stages, with plans to incorporate investor sentiment data from Reddit and other social media platforms to enhance the model's predictive power. By integrating sentiment analysis, StockVision aims to capture market sentiment and investor behavior, offering a more comprehensive view of stock price movements. Due to a lack of availability of data, it is difficult to accurate predict stock prices as they fluctuate based on a variety of live factors.
    The next steps for this project are to:
    1. Implement a more robust sentiment analysis model to capture investor sentiment from social media platforms.
    2. Ensure the model is able to handle this live data and display predictions in real-time to users on streamlit.

    Gravitational Simulation

    Players can initialize different velocities and directions and the program will automatically calculate and blit their movement onto the screen. The program automatically calculates and adjusts the velocity of the dot as it approaches earth and stops it from "crashing" by automatically putting it into orbit. This is not true for the moon, which "ships" can actually crash into. The velocities and accelerations are blitted to the screen as well as the # of active "spacecraft".
